Browser API Sandbox
==================

# About

This a sandbox of various examples on how to use some browser apis. If you have any useful snippets, feel free to fork + PR.

Current examples :
  - geolocation
  - battery
  - bluetooth
  - server
  - web audio (creation / analysis)
  - accelerometer
